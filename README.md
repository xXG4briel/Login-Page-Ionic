## Página de Login Angular/Ionic

**Requisitos**
- Ionic
- Angular
- Node

**Passo a Passo**

- Criando o projeto
`
ionic start nomeDoProjeto blank
`


- Entrado dentro do diretório criado
`
cd ./nomeDoProjeto
`


- Iniciando o servidor
`
ionic serve
`
